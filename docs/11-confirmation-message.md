# Node 11 — Confirmation Message

## Objetivo

Enviar a mensagem final de confirmação no WhatsApp.

## Entrada

- dados do evento criado
- serviço
- data
- hora
- local
- lembretes configurados

## Processamento

- montar mensagem em português-BR
- aplicar formatação compatível com WhatsApp
- reforçar próximos passos

## Saída

- payload de mensagem de confirmação

## Regras críticas

- usar tom humano e coerente com o personagem
- mencionar lembretes e canal de retorno

## Dependências externas

- WhatsApp Business API

## Observações de implementação

A mensagem deve ser curta, clara e segura para leitura em celular.
