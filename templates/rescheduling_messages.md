# Rescheduling Messages

## Cenário: iniciar reagendamento

**Mensagem base**

`Claro, vamos remarcar seu atendimento. Vou verificar novos horários para você agora.`

**Placeholders**
- nenhum obrigatório

**Uso no fluxo**
- Node 14

## Cenário: novo horário confirmado

**Mensagem base**

`Pronto, [NOME]! Seu novo horário para *[SERVIÇO]* ficou em [DATA] às [HORA]. Se precisar de algo, é só me chamar.`

**Placeholders**
- `[NOME]`
- `[SERVIÇO]`
- `[DATA]`
- `[HORA]`

**Uso no fluxo**
- Node 14 + Node 11
