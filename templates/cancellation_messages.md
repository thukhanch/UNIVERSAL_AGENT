# Cancellation Messages

## Cenário: cancelamento confirmado

**Mensagem base**

`Cancelamento confirmado, [NOME]. Sentimos muito que não vai rolar dessa vez. Quando quiser marcar novamente, é só me chamar aqui.`

**Placeholders**
- `[NOME]`

**Uso no fluxo**
- Node 13

**Mensagem base**

`Antes de concluir, vale lembrar nossa política: {{POLITICA_CANCELAMENTO}}.`

**Placeholders**
- `{{POLITICA_CANCELAMENTO}}`

**Uso no fluxo**
- Node 13
