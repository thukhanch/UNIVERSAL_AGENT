# Variables Template

Preencha apenas estas variáveis para adaptar o agente a qualquer negócio.

```txt
{{NOME_EMPRESA}}=
{{SEGMENTO}}=
{{NOME_AGENTE}}=
{{PERSONAGEM}}=
{{MISSAO}}=
{{FILOSOFIA}}=
{{TOM_DE_VOZ}}=
{{ESTILO_COMUNICACAO}}=
{{FRASE_EXEMPLO}}=
{{SERVICOS_DISPONIVEIS}}=
{{DURACAO_SERVICOS}}=
{{HORARIO_FUNCIONAMENTO}}=
{{FUSO_HORARIO}}=
{{TELEFONE_HUMANO}}=
{{LINK_PAGAMENTO}}=
{{POLITICA_CANCELAMENTO}}=
{{LEMBRETE_ANTECEDENCIA_1}}=
{{LEMBRETE_ANTECEDENCIA_2}}=
{{MENSAGEM_BOAS_VINDAS}}=
{{BASE_DE_CONHECIMENTO}}=
{{CALENDARIO_ID}}=
{{GOOGLE_API_KEY}}=
{{EMOJIS_PERMITIDOS}}=
```

## Orientações

- Preencha tudo antes do deploy.
- Se um campo não for utilizado, escreva `não utilizado`.
- Mantenha nomes e serviços exatamente como serão usados em operação.
- Em `DURACAO_SERVICOS`, prefira o padrão `Serviço=60min`.
