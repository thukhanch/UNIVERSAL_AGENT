# Playground

Frontend estático simples para testar o `mock-backend` local.

## Como usar

1. Inicie o backend:
   - `cd mock-backend`
   - `npm start`
2. Abra `playground/index.html` no navegador.
3. Use os botões rápidos ou escreva sua própria mensagem.
4. Veja o JSON de resposta retornado pelo backend.

## Cenários rápidos

- Agendamento
- Cancelamento
- Reembolso
- Reclamação
- Áudio
